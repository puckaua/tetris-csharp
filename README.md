# Tetris C#
Versão simples do clássico jogo Tetris, executando diretamente no Console.

O jogo contém o básico que funciona muito bem: 3 tipos de peças diferentes, podendo ser giradas no sentido horário e anti-horário.
Além disso, salva as pontuações do jogador em um arquivo texto no diretório corrente.

Comandos:
- mover peça para baixo (nessa versão, o jogador move a peça para baixo): seta para baixo (down arrow key);
- mover peça para a esquerda: seta para a esquerda (left arrow key);
- mover peça para a esquerda: seta para a direita (right arrow key);
- girar a peça 90° para a esquerda: z;
- girar a peça 90° para a direita: x;

  O jogo é finalizado quando as peças ocupam uma coluna inteira do tabuleiro, tocando o topo.

  # Como baixar
  (Necessário ter instalado .NET Framework v4.8. Caso não tenha, <a href="https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48">clique aqui para fazer o download</a>)
  1 - Vá na pasta installer e baixe o arquivo TetrisInstaller.exe; <br>
  2 - Execute o instalador que acabou de baixar e finalize a instalação; <br>
  3 - Um atalho será adicionado à sua área de trabalho, basta abri-lo para começar a jogar. <br>
  4 - Divirta-se!<br>

   # Como baixar o código-fonte
  1 - Baixar a pasta compactada em Code > download zip; <br>
  2 - Extrair o conteúdo da pasta compactada; <br>
  3 - Abrir o Visual Studio e selecionar "abrir uma pasta local"; <br>
  4 - Selecionar a pasta tetris-csharp-main; <br>
  5 - Na parte superior central, selecione o item de inicialização tetris-csharp.csproj; <br>
